# How to Start a Startup — Claude Mentor Skill

A Claude skill built from the full Stanford CS183B course. Get startup advice from the frameworks of Sam Altman, Peter Thiel, Paul Graham, Reid Hoffman, Ben Horowitz, and 17 others — applied to your specific idea.

## How to use

1. Go to [Claude.ai](https://claude.ai) and open a **Project**
2. In Project Instructions, paste the contents of `SKILL.md`
3. Start a new chat and type `/how-to-start-a-startup-mentor`
4. Claude will run an intake questionnaire, then give you direct, cited advice based on your situation

## What it does

- Asks the exact questions a YC partner would ask
- Applies frameworks from all 20 lectures to your specific idea
- Cites which speaker and lecture every piece of advice comes from
- No generic responses — everything is mapped to your context
